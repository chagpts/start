아래 카드를 옆으로 넘기면서 각 교육 모듈의 내용을 확인해보세요.

Module 01

Accounting Information System

회계 데이터가 실제 시스템 안에서 어떻게 입력, 저장, 처리되는지 이해합니다.

  • 회계 전산 시스템 구조 이해
  • 거래 입력과 전표 처리 흐름
  • 계정과목, 장부, 재무제표 연결
  • 실무형 회계 데이터 구조 실습
Module 02

AI OCR System

영수증, 인보이스, 증빙자료를 AI OCR로 읽고 회계 데이터로 변환하는 과정을 실습합니다.

  • AI OCR 기본 원리 이해
  • 영수증 및 인보이스 인식
  • 자동 분개 데이터 생성
  • 오류 검토 및 보정 방법
Module 03

Audit & ITGC

내부통제와 IT General Control을 이해하고 데이터 기반 감사 절차를 학습합니다.

  • ITGC 핵심 개념
  • 접근권한 통제
  • 변경관리 및 운영통제
  • 감사 증적 확인 실습
Module 04

Vulnerability Analysis and Defense

회계 시스템과 데이터 처리 환경에서 발생할 수 있는 취약점을 분석하고 방어 방법을 배웁니다.

  • 시스템 취약점 기본 이해
  • 데이터 보안 위험 분석
  • 권한 관리 취약점 점검
  • 보안 개선 방안 수립
Module 05

Real Case Practice

실제와 유사한 회계 데이터를 기반으로 AI, 감사, 내부통제를 연결한 실습을 진행합니다.

  • 샘플 회계 데이터 분석
  • 이상 거래 탐지
  • 감사 체크리스트 작성
  • 결과 리포트 작성
Module 06

Internal Audit and Career Development

전산 감사, 내부회계, IT Audit, AI 회계 자동화 분야로 커리어를 확장하는 방법을 다룹니다.

  • 전산 감사 직무 이해
  • 필요 역량과 포트폴리오
  • 실무 프로젝트 구성 방법
  • 커리어 방향 설정
