What is AWS?

AWS (Amazon Web Services) is a cloud computing platform provided by Amazon. Instead of buying physical servers, companies can rent computing resources using the internet.

  • servers
  • storage
  • databases
  • networking
  • AI tools

AWS is widely used because it helps businesses build and run services faster, scale more easily, and reduce the burden of managing hardware.

Part 2 – Core Reading

Reading Passage

AWS provides on-demand IT resources through a pay-as-you-go model.

For example, if a company needs a server for only one week, they can rent it from AWS and shut it down later. They do not need to maintain hardware.

AWS also offers global infrastructure. Companies can run applications in many regions around the world to reduce latency and improve user experience.

Security and scalability are major benefits. Businesses can grow quickly without rebuilding their systems.

Comprehension Questions

  • What payment model does AWS use?
  • Why is global infrastructure important?
  • What are two major benefits mentioned?

Part 3 – Key Services Vocabulary

Category Service / Term Meaning (Simple English) Example Sentence
Compute EC2 (virtual servers) Rent a server to run apps We use EC2 to run our web application.
Storage S3 (object storage) Store files and data S3 is good for storing images and backups.
Database RDS (managed database) A database you don’t manage yourself RDS helps us manage a relational database easily.
Networking VPC / Load Balancer / CDN Connect services and handle traffic A load balancer distributes traffic to multiple servers.

Part 5 – Dialogue Practice

Situation: A manager asks why the team should move to AWS.

Manager (Finance): Why are you auditing our AWS environment?

IT Auditor: Because your financial data is generated from systems hosted in AWS. We need to verify the reliability of the controls.

Manager: But the numbers look correct. Why is this necessary?

IT Auditor: Even if the numbers look fine, weak access or change controls can make the reports unreliable.

Manager: What areas are you focusing on?

IT Auditor: We are reviewing user access, change management, backups, and security monitoring.

Key Expressions

  • We need audit evidence.
  • We must rely on system controls.
  • Who has access to production?
  • Is there an approval record?
